Projeto para treinar efeito parallax em imagens.

PROJETO DE REDES SOCIAIS

Projeto para demonstração de responsividade e múltiplas media queries.

Para este projeto, decidi levar a responsividade a sério. O grande astro aqui foi o uso de Media Queries! Mais um passo na minha jornada de dominar o desenvolvimento responsivo!

PROJETO TELA DE LOGIN

Projeto para treinar a responsividade de formulários com uma tela de login.

Este projeto consiste em uma página de login moderna desenvolvida com foco em UI Design e experiência do usuário. Utilizei CSS Flexbox/Grid para a estruturação e técnicas de Glassmorphism e efeitos de brilho externo (box-shadow glow) para criar uma estética vibrante. A interface conta com validação visual de campos, ícones integrados aos inputs e total adaptabilidade para diferentes tamanhos de tela.

O foco foi em UI e UX, garantindo que as imagens compusessem uma narrativa visual coesa e impactante.
